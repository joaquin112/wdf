My Web Standards

Day 5 - February 9, 2012

I liked today's lecture. It had a lot of information about tables and tables inside tables. I learned not to use tables for layout because those layouts tend to look really ugly.

An example of a tag I used is the <table>, which allows you to start a table. This tag is really useful and it's used in conjunction with <tr>, <td>; and a few other tags that were discussed during lecture.
