Validation of Four Websites

Youtube

Youtube is the most popular video-sharing website.

Section 508 Validation

  1. Missing alt attributes: The developer should add all the appropriate attributes.
  2. Clearly identify the target of each link: They should add a clear title for each link.
  3. Associate labels explicitly with their controls: They should create more labels for the content.
  4. Validate that Alternative Text is greater than 7 and less than 81 characters in length: Youtube isn't using any characters in most alt attributes.

WCAG Validation

  1. A text equivalent for every non-text element shall be provided (e.g., via "alt", "longdesc", or in element content): They failed. There are more than 30 elements which don't have text alternative content.
  2. Validate that Alternative Text is not used to repeat words: Some words are repeated.

Godaddy.com

Godaddy is the most popular domain name registrar.

Section 508 Validation

  1. Validate that Alternative Text is greater than 7 and less than 81 characters in length: Godaddy should put an alt tag in all its elements and be more than 7 characters and less than 81 characters.

WCAG Validation

  1. A text equivalent for every non-text element shall be provided (e.g., via "alt", "longdesc", or in element content): This is the same as in 508 - what they should basicall ydo is add more alt tags.
