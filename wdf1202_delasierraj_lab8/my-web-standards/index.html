My Web Standards

Day 8 - February 16, 2012

Today's class focused on usability. We learned many different and cool things related to making our websites usable for all sorts of people. I found this class very interesting because we learned that making a website usable not only helps it be more friendly for disabled users, but for all users in general. I think that this is a very important concept to understand.

We also learned how to make tables more accessible for all kinds of users. This is done by using a few new tags that we learned today. All in all, it was a good class. The element I liked the most, though was the "label" element, because it helps forms be much easier to use.

Day 7 - February 14, 2012

In today's lecture we learned how to use XML files. We learned that they are used to store data or to share data to another application such as a Flash application. I think that's very cool because you can easily share data with other applications and that makes it easy to use and effective..

Additionally, we learned about RSS feeds which are pretty nice to use as they allow people to receive information every day from many different sources.

Useful link of the day: RSS Feeds

Day 6 - February 11, 2012

In today's lecture we learned how to make forms using several different HTML elements. We learned how to use the input type element, how to use textarea and a lot of other cool things.

I liked today's lecture because we also learned how to input data to another website such as Google or Wikipedia. For example, it's possible to send data to Wikipedia and show the user one of Wikipedia's pages.

Useful link of the day: HTML Forms

Day 5 - February 9, 2012

I liked today's lecture. It had a lot of information about tables and tables inside tables. I learned not to use tables for layout because those layouts tend to look really ugly.

An example of a tag I used is the <table>, which allows you to start a table. This tag is really useful and it's used in conjunction with <tr>, <td>; and a few other tags that were discussed during lecture.

Useful link of the day: HTML Tables
