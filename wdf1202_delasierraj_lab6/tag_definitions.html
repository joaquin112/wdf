Tag Definitions

Definitions

p
- Start of paragraph
h1
- Header (Large)
h2
- Header (Medium)
h3
- Header (Small)
ul
- Unordered list
ol
- Ordered HTML list
li
- Individual items
em
strong
This makes the text stronger, emphazises it.
abbr
This signals an abbreviation.
acronym
Signals an acronym to the browser.
cite
This is used for making a citation
code
This is used to display code.
kbd
Used to display keyboard keys.
samp
Uses to output an example of something.
var
It's used to indicate a variable
del
Del is used to scratch through text.
ins
Used to show text inserted into a document
br
This is a line break
hr
This is a horizontal line
pre
Pre is used to display the text "as is"
sup
Displays superscript text.
sub
Displays subscript text.
address
Used to display the address of someone.
dfn
Used to indicate a definition.
table
Tables are used to display data in an ordered way. They should not be used for layouts.
tr
The tr tag is used to create different rows inside a table.
th
The th tag is used to mention the header of a table, so that the tags are in bold.
td
The td tag is used to create new cells.
thead
The tdhead tag is used to let the browser know that this part of the table is the head of the table.
tfoot
The tfoot tag is used to specify what the footer of the table is.
tbody
The tbody tag is used to specify the content of the table.
Form
The form tag is used to enclose form elements and define where the data should be submitted and how.
Inout
The input tag is used to signal the browser that a new form field should be created. It allows several attributes such as:
Text
The text attribute is used to allow users to enter their information.
Checkbox
The checkbox attribute is used to create a checkbox and allows the user to select between multiple choices.
Radio
The radio attribute lets the user select among many options, but it only allows for one selection.
Submit
The submit attribute is used to display an image or button and submit the form.
Reset
The reset attribute is used to reset all the information filled inside a form by the user.
The TEXTAREA Tag
The TEXTAREA tag is used to create a form input field that allows text for several lines.
Select
The select tag is used to enclose several option tags and create a dropdown field in the form.
Option
The option tag is used to specify different elements within the dropdown specified by the select enclosing tag.
