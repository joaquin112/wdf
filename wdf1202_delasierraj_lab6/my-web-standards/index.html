My Web Standards

Day 6 - February 11, 2012

In today's lecture we learned how to make forms using several different HTML elements. We learned how to use the input type element, how to use textarea and a lot of other cool things.

I liked today's lecture because we also learned how to input data to another website such as Google or Wikipedia. For example, it's possible to send data to Wikipedia and show the user one of Wikipedia's pages.

Useful link of the day: HTML Forms

Day 5 - February 9, 2012

I liked today's lecture. It had a lot of information about tables and tables inside tables. I learned not to use tables for layout because those layouts tend to look really ugly.

An example of a tag I used is the <table>, which allows you to start a table. This tag is really useful and it's used in conjunction with <tr>, <td>; and a few other tags that were discussed during lecture.

Useful link of the day: HTML Tables
