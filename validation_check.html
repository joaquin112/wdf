This is the Validation Check

This webpage will show the validation errors of some popular websites.

Youtube

Youtube is a website that streams videos to its users. Youtube has 177 errors.

  • Error 1: Bad value for attribute id on element body - A value should be written in the id element.
  • Bad value submit for attribute type on element button. This attribute should be changed to meet the guidelines
  • The scrolling attribute on the iframe element is obsolete. Use CSS instead. - They should update their current HTML to support more CSS.

9Gag.com

9Gag.com is a website with funny pictures.

  • there is no attribute "prefix" - The designer should change the name of this attribute. The current attribute doesn't exist.
  • there is no attribute "property" - The designer should change the name of the "property" attribute as it doesn't work with XHTML strict 1./
  • there is no attribute "placeholder" - This attribute doesn't exist.

W3.org

This is the official website of the W3. They set the standards for web developers.

Luckily, there are no errors in this page

Netflix.com

Netflix is a service that streams movies to users.

  • required attribute "TYPE" not specified - The developer should include the Type definition for this element.
  • document type does not allow element "STYLE" here - The developer should not have included this element in this section
  • document type does not allow element "DIV" here - There shouldn't be a div here. The designer should remove it
